# opengl_Renderer
Simple renderer for OpenGL

## DESCRIPTION

Dependencies:

* GLEW
* GLFW
* glm (OpenGL Mathematics)
* SOIL(png,jpeg,tga...load)
* ASSIMP(Model Load)
* Freetype

## Features

* Shader based (GLSL)
* Component based scene graph
* Model loading (ASSIMP)
* font draw
